GET

Generate a Sudoku

Creates a new Sudoku puzzle based on difficulty constraints and a target clue count.

?method=generate&min={float}&max={float}&clues={int}
Parameter Type Description
minFloatMinimum difficulty (0.0 - 1.0)
maxFloatMaximum difficulty (0.0 - 1.0)
cluesIntegerNumber of visible cells in the starting grid
GET

Solve a Sudoku

Calculates the full solution for a given Sudoku board string.

?method=solve&string={81_char_string}
GET

Rate Difficulty

Analyzes the logic required to solve the puzzle and returns a rating.

Rating Key:

  • 0.0 to 1.0: Standard difficulty.
  • -0.1: Too complex / Advanced techniques required.
  • -0.2: Board contradiction (Invalid).
GET

Get a Hint

Returns a single logical step to help move forward from the current board state.

Sudoku of the Day

Retrieves a curated puzzle assigned to a specific date.

?method=daily&day=2026-03-30

Board Data Format

The API uses a standard 81-character string format to represent the grid:

EXAMPLE STRING
010000602906523871025006003600905147504617230730200569200309000163002980470851320
  • Read row-by-row, from top-left to bottom-right.
  • Use 0 to represent an empty cell.
