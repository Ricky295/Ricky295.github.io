🎲 Generate

Standard Generation Updated

?method=generate&min=0&max=0.33&clues=50&count=1

Generate one or more Sudoku puzzles within a difficulty range.

min (float, default: 0.0) — Minimum difficulty
max (float, default: 0.33) — Maximum difficulty
clues (int, default: 50) — Target number of clues
count (int, default: 1) — Number of puzzles to generate
Request
?method=generate&min=0.33&max=0.67&clues=40&count=5
Response
{ "puzzles": ["530070000...", "..."], "count": 5 }

Impossible Generation New

?method=generate&impossible=1&clues=17&count=1

Generate one or more puzzles unsolvable by standard techniques (rates as -0.1). Finds minimal clues automatically.

impossible (flag: 1) — Generate "impossible" puzzle
clues (int, default: 17) — Target clues (minimum recommended: 17)
count (int, default: 1) — Number of puzzles to generate
Request
?method=generate&impossible=1&clues=20&count=3
Response
{ "puzzles": ["530070000...", "..."], "count": 3, "difficulty": -0.1 }

Seed-Based Generation New

?method=seedgen&seed=mystring

Generate a reproducible puzzle using only a seed string. Difficulty varies based on seed.

seed (string, required) — Seed for reproducibility
Request
?method=seedgen&seed=april2026
Response
{ "puzzle": "530070000...", "seed": "april2026" }

Daily Puzzle

?method=daily&day=2026-04-21

Get the daily Sudoku for a specific date (deterministic).

day (YYYY-MM-DD, optional) — Date (defaults to today)
Request
?method=daily&day=2026-04-21

🔧 Solve

Basic Solve

?method=solve&puzzle=530070000...

Solve using logical techniques only (stops if unsolvable by techniques).

puzzle (string, required) — 81-character Sudoku string

Full Solve New

?method=solve&puzzle=...&full_solve=1

Solve using logic, then complete with backtracking if needed.

puzzle (string, required) — 81-character Sudoku string
full_solve (flag: 1) — Enable full completion

Solution Path New

?method=solve_path&puzzle=...

Get step-by-step solving instructions with techniques used.

puzzle (string, required) — 81-character Sudoku string
Response
{ "steps": [ { "technique": "Box HS", "row": 0, "col": 1, "value": 4 }, ... ] }

📊 Analysis

Rate Difficulty Updated

?method=rate&puzzle=...

Analyze puzzle difficulty and validate solvability.

puzzle (string, required) — 81-character Sudoku string
Response Values
0.0 – 1.0 Valid puzzle with difficulty rating
-0.1 Unsolvable by techniques (requires guessing)
-0.2 Contradiction detected (impossible puzzle)
-0.3 Multiple solutions (invalid puzzle)

Get Hint

?method=hint&puzzle=...

Get the next solving step with technique and location.

puzzle (string, required) — 81-character Sudoku string
Response
{ "technique": "Box HS", "row": 2, "col": 4, "value": 7 }

📖 Reference

Puzzle Format

Puzzles are represented as 81-character strings (rows 0–8, left to right). Use 0 for empty cells.

5 3 0 | 0 7 0 | 0 0 0
6 0 0 | 1 9 5 | 0 0 0
0 9 8 | 0 0 0 | 0 6 0
------+-------+------
8 0 0 | 0 6 0 | 0 0 3
4 0 0 | 8 0 3 | 0 0 1
7 0 0 | 0 2 0 | 0 0 6
------+-------+------
0 6 0 | 0 0 0 | 2 8 0
0 0 0 | 4 1 9 | 0 0 5
0 0 0 | 0 8 0 | 0 7 9

Difficulty Scale

Range Level Techniques
0.0 – 0.33 Easy Box Hidden Singles
0.33 – 0.67 Medium Box & Line Hidden Singles
0.67 – 1.0 Hard + Naked Singles
-0.1 Impossible Beyond standard techniques

Solving Techniques

Technique Description
Naked Single Cell with only one possible value
Box HS Value that can only go in one cell within a 3×3 box
Line HS Value that can only go in one cell within a row or column
