Sudoku API Guide

How to use it

Every call is a GET request with a method query parameter. Responses are JSON wrapped in { status, data }. No auth, no headers, no request body — just build a URL and fetch it.

1

Get a puzzle

Call ?method=seedgen&seed=yourstring for a reproducible puzzle, or ?method=generate&min=0&max=0.33&clues=45 to target a difficulty band. Both return an 81-character puzzle string.

2

Get the solution

Pass that puzzle string to ?method=solve&puzzle=...&full_solve=1. The full_solve flag guarantees a complete grid — without it, solving stops wherever logical techniques run out.

3

Check its difficulty

Call ?method=rate&puzzle=... any time — on a fresh puzzle or the player's current progress — to get a difficulty score back.

4

Offer a hint mid-game

Send the player's current board state (not the original puzzle) to ?method=hint&puzzle=.... It returns the next logical move and which technique produces it.

Generate

Three ways to get a new puzzle: by difficulty range, deterministically from a seed, or as today's daily.

Standard generation

?method=generate&min=0&max=0.33&clues=45&count=1

Generates a puzzle whose rated difficulty falls inside min–max, with roughly clues starting numbers.

min — float, default 0.0. Lower bound on difficulty.
max — float, default 0.33. Upper bound on difficulty.
clues — int, default 50. Target starting clue count.
count — int, default 1. Number of puzzles to return.
Response
{ "status": "success", "data": { "puzzle": "038900160..." } }

Impossible generation Rates as −0.1

?method=generate&impossible=1&clues=17

Generates a puzzle that can't be finished with the logical techniques this engine knows — it needs guessing (backtracking) to complete. Clue count is pushed as low as possible automatically.

impossible — flag, set to 1.
clues — int, default 17. Starting target; the engine tries to remove more if possible.

Seed-based generation

?method=seedgen&seed=mystring

Same seed always returns the same puzzle. Difficulty isn't targeted — it falls out of whatever the seed produces.

seed — string, required.

Daily puzzle

?method=daily&day=2026-04-21

Deterministic puzzle for a given calendar date, in the easy-to-medium range. Defaults to today if day is omitted.

day — YYYY-MM-DD, optional.

Solve

Two depths of solving, plus a step log if you want to show your work.

Solve (logic only)

?method=solve&puzzle=038900160...

Applies logical techniques until it either finishes the grid or runs out of moves. If the puzzle needs guessing, the result comes back incomplete.

puzzle — 81-char string, required.

Solve (full)

?method=solve&puzzle=...&full_solve=1

Same as above, then falls back to backtracking to guarantee a complete solution. This is what you want when you just need the answer key.

full_solve — flag, set to 1.

Solution path

?method=solve_path&puzzle=...

Returns every logical step in order — technique, cell, and value — useful for building an explainable solve animation.

Response
{ "steps": [ { "technique": "BoxHS", "row": 0, "col": 1, "value": 4, "remaining": 39 }, ... ] }

Rate

Score any puzzle state — a fresh puzzle, a partially solved board, whatever's on the grid.

?method=rate&puzzle=...

Runs the same logical solving pass used internally by generate, and returns how hard it was.

puzzle — 81-char string, required.
difficulty = ((steps / initialEmpty) + highestTechnique) / 3

Where steps is the number of solving passes needed, initialEmpty is how many cells started blank, and highestTechnique is the hardest technique the solve required (0 = box hidden single, 1 = line hidden single, 2 = naked single).

ScoreMeaning
0.0 – 1.0Valid puzzle, difficulty as computed above
−0.1Unsolvable by known techniques — needs guessing
−0.2Contradiction — puzzle has no valid solution
−0.3Multiple solutions — puzzle is under-constrained

Hint

One move at a time, in order of simplicity.

?method=hint&puzzle=...

Checks for a box hidden single first, then a row/column hidden single, then a naked single, and returns the first one it finds.

Response
{ "technique": "Box HS", "row": 2, "col": 4, "value": 7 }
