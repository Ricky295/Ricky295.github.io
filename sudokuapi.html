Sudoku Engine API

A robust RESTful API for generating, solving, and analyzing Sudoku puzzles. Built with Google Apps Script, featuring logical solving techniques, seeded generation for reproducibility, and difficulty rating.

Base URL: https://script.google.com/macros/s/AKfycbzVzqnvdupVkMH0gs9XUGefJoCcUz3PjcCT6oMRXtlZC82LN9pOlXXIJDiEXuLquZRq/exec

Generate Puzzle

GET Generate Random Puzzle

Generates a valid Sudoku puzzle within a specific difficulty range.

Parameters

Param Type Default Description
method string - Must be generate
min float 0.0 Minimum difficulty rating
max float 0.33 Maximum difficulty rating
clues int 50 Target number of starting clues
impossible int 0 Set to 1 to generate an unsolvable puzzle

Example Request

?method=generate&min=0.1&max=0.4&clues=30

Response

{
  "status": "success",
  "data": {
    "puzzle": "000000000000000000..." 
  }
}

Seeded Generation

GET Generate Deterministic Puzzle

Generates a puzzle using a seed string. The same seed will always produce the same puzzle and the same random clue count (0-50).

Parameters

Param Type Description
method string Must be seedgen
seed string Any string used as the random seed

Example Request

?method=seedgen&seed=my-daily-challenge

Response

{
  "status": "success",
  "data": {
    "puzzle": "400000201...",
    "seed": "my-daily-challenge",
    "clues": 27
  }
}

Solver

Solve Puzzle

Returns the completed solution for a given puzzle string.

?method=solve&puzzle=...&full_solve=1
  • puzzle: 81-character string (0 for empty)
  • full_solve: Optional. Set to 1 to use backtracking if logical steps fail.

Solve Path

Returns a step-by-step logical deduction path. Useful for teaching or hint systems.

?method=solve_path&puzzle=...

Response Example

{
  "steps": [
    { "technique": "BoxHS", "row": 1, "col": 0, "value": 1, "remaining": 50 },
    { "technique": "NakedSingle", "row": 2, "col": 2, "value": 5, "remaining": 49 }
  ],
  "total_steps": 51
}

Analysis Tools

Rate Difficulty

Calculates the difficulty score based on logical techniques required.

?method=rate&puzzle=...

Formula: ((steps / initialEmpty) + highestTechnique) / 3

Get Hint

Provides the next logical move without solving the whole puzzle.

?method=hint&puzzle=...

Response

{
  "technique": "Line HS",
  "row": 4,
  "col": 5,
  "value": 9
}

Daily Challenge

GET Get Daily Puzzle

Generates a consistent puzzle for a specific date. If no date is provided, uses today's date.

?method=daily&day=2026-07-08

Score Range Meaning Color Code
1.0 Valid, Solvable Green
-0.1 Unsolvable by basic techniques Yellow
-0.2 Contradiction (Invalid Board) Red
-0.3 Multiple Solutions Red
