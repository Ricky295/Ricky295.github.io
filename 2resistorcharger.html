Thevenin Divider Equations Reference

1. Thevenin Source (R_th)

Computes internal resistance needed to limit initial charging current ($I_{\text{max}}$) at safe minimum dead battery threshold ($V_{\text{dead}}$).

R_th = (V_charge - V_dead) / I_max
2. Upstream Resistor (R1)

Matches source input drops to safely scale the charging network to peak target voltage requirements.

R1 = R_th * (V_in / V_charge)
3. Ground Resistor (R2)

Provides the stable passive pull-down ensuring strict voltage clamping to secure against cell overcharge risks.

R2 = R_th * (V_in / (V_in - V_charge))
