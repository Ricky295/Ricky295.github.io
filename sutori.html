Sutori

New Puzzle Variant! Sutori uses a 12×12 grid with 4×3 blocks. Numbers 1-4 must appear exactly 3 times in each row, column, and block, with no same digits touching orthogonally. Difficulty is currently clue-based as we develop proper difficulty algorithms.

Rules: Fill the 12×12 grid so that each row, column, and 4×3 rectangle contains the numbers 1-4 exactly three times each. No same numbers may touch orthogonally (horizontally or vertically adjacent).
