State of Charge (SoC) Calculation

Calculate remaining energy capacity percentages

GET method=percentage

Required Request Arguments

Parameter Type Requirement Details
chemistry string Required The battery chemistry alias (e.g. `lipo`, `lfp`, `na`).
voltage float Required Absolute measured total pack voltage.
cells integer Optional Series cell count multiplier. Defaults to `1`.
Underlying SoC Mathematical Curves

The API calculates the single-cell voltage $v_{cell} = \frac{\text{voltage}}{\text{cells}}$ and evaluates optimized curves. For example, the Lithium-Ion / Polymer piecewise profile scales as:

For $v_{cell} > 3.4\text{V}$:

$\text{SoC} = (112.5 \cdot v_{cell}) - 372.5$

For $3.0\text{V} \le v_{cell} \le 3.4\text{V}$:

$\text{SoC} = 10 \cdot \left(\frac{v_{cell} - 3.0}{3.4 - 3.0}\right)^3$

State of Health (SoH) Calculation

Evaluate structural degradation and resistance decay

GET method=health

Required Request Arguments

Parameter Type Requirement Details
chemistry string Required The battery chemistry key/alias (e.g. `na-ion`, `lfp`).
ir float Required Measured internal resistance (IR) in milliohms ($\text{m}\Omega$).
SoH Decay Exponential Slope Modeling

State of Health uses an exponential degradation curve to calculate structural losses based on factory configurations.

Given a chemistry's fresh baseline $x_1$, warning resistance limit $x_2$, and degradation scale $y_2$:

1. Calculate logarithmic decay factor ($b$):

$b = \frac{-\ln(y_2)}{x_2 - x_1}$

2. Solve for active SoH Percentage:

$\text{SoH (\%)} = 100 \cdot e^{-b \cdot (\text{ir} - x_1)}$

Language Integrations

const getBatterySoC = async (chemistry, voltage, cells = 1) => {
  const baseUrl = "https://script.google.com/macros/s/AKfycbwEtsSR4ke-qrwIgfCMd29a85mpkeLmXFEJbTpv0VKKNBhZwxJjAWYVVLWDJWIVL1livA/exec";
  const url = `${baseUrl}?method=percentage&chemistry=${encodeURIComponent(chemistry)}&voltage=${voltage}&cells=${cells}`;
  
  try {
    const response = await fetch(url);
    const data = await response.json();
    if (data.status === "success") {
      console.log(`Battery SoC: ${data.state_of_charge_pct}%`);
      return data;
    } else {
      console.error("API Error:", data.message);
    }
  } catch (err) {
    console.error("Network Error:", err);
  }
};

// Example Usage
getBatterySoC("lithium polymer", 12.0, 3);
import requests

def get_battery_health(chemistry, ir_mohm):
    base_url = "https://script.google.com/macros/s/AKfycbwEtsSR4ke-qrwIgfCMd29a85mpkeLmXFEJbTpv0VKKNBhZwxJjAWYVVLWDJWIVL1livA/exec"
    params = {
        "method": "health",
        "chemistry": chemistry,
        "ir": ir_mohm
    }
    
    try:
        response = requests.get(base_url, params=params)
        response.raise_for_status()
        data = response.json()
        if data.get("status") == "success":
            print(f"Battery SoH: {data['state_of_health_pct']}%")
            return data
        else:
            print(f"API Error: {data.get('message')}")
    except requests.exceptions.RequestException as e:
        print(f"Network error: {e}")

# Example Usage
get_battery_health("sodium ion", 150)
